## Iconography


## Design specifications

All official SVG icons are available as major icons.

## Icon dimensions

While it is possible to adjust both the width and height dimensions, it is recommended to maintain a 1:1 aspect ratio, resembling a square shape, for optimal visual presentation. This ensures that the icons retain their intended appearance and aesthetic quality.

#### Usage

```jsx
import { AbacusIcon } from "@afrodita/ui-kit-t1";

function render() {
  return <AbacusIcon width={24} height={24} />;
}
```

### Static files

Icons SVG files are available from the root of `@afrodita/ui-kit-t1` package. These files are useful for non React apps or any use cases which cannot be done using a component. Whenever you can, you should prefer React components to static files.

Although frowned upon, you can customize an icon dimensions with the `width` and `height`.

#### Image source

```jsx
import CalendarMajorIcon from "@afrodita/ui-kit-t1/icon-calendar-major.svg";

function render() {
  return <Img src={CalendarMajorIcon24} alt="Calendar" width={24} height={24} />;
}
```

## Color

An SVG object get it's color from the [fill attribute](https://developer.mozilla.org/en-US/docs/Web/SVG/Tutorial/Fills_and_Strokes). When an SVG object doesn't define a `fill` attribute on his elements, CSS can be use to style the internal color of the elements of the objects with the `fill` property.

This means you can alter the color of an icon by specifying a new `fill` value. If the SVG already contains a `fill` attribute.

### React components

To set the color of an icon component, you can set the `color` prop.

```jsx
import { AbacusIcon } from "@afrodita/ui-kit-t1/src/assets/icons";

function render() {
  return <AbacusIcon color="red" width={24} height={24} />;
}
```

### Image source

To set the color of an image, you can set the `fill` prop.

```jsx
import CalendarMajorIcon from "@afrodita/ui-kit-t1/icon-calendar-major.svg";

function render() {
  return <Img src={CalendarMajorIcon} alt="calendar" fill="purple-8" />;
}
```
